Set the pointer data associated with one attributes.

If the shader program is invalid no error will be returnes and the shader program will not be modified

After successfully calling this function a call to glDrawArrays or glDrawElements with this shader program being bound will use this data

This function is mostly used internally

Parameters

componentsThe number of components in the attribute
typeThe type of data component (e.g. GL_FLOAT)
normalizeSpecify if the data should be normalized when accessed
pointerA pointer to the data
strideByte offset between the beginning of two attributes (0 means tightly packed)
Returns
The error-state of the function
