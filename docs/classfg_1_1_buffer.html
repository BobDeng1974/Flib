Member Enumeration Documentation

◆ Usage

enum fg::Buffer::Usage

Values used to hint OpenGL on the usage of the data.

Enumerator
StaticDraw 

The application will send data to OpenGL once (Please note that this is only a hint)

StaticRead 

OpenGL will send data to the application once (Please note that this is only a hint)

StaticCopy 

Data will be transferred inside OpenGL once (Please note that this is only a hint)

DynamicDraw 

The application will send data to OpenGL frequently (Please note that this is only a hint)

DynamicRead 

OpenGL will send data to the application frequently (Please note that this is only a hint)

DynamicCopy 

Data will be transferred inside OpenGL frequently (Please note that this is only a hint)

StreamDraw 

The application will send data to OpenGL per frame (Please note that this is only a hint)

StreamRead 

OpenGL will send data to the application per frame (Please note that this is only a hint)

StreamCopy 

Data will be transferred inside OpenGL per frame (Please note that this is only a hint)
