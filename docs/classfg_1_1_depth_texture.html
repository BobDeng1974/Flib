Detailed Description

A special texture used to hold depth values.

A depth texture treats its pixels as if they were one unsigned integer each

fg::DepthTexture::DepthTexture ( const Image &  img)
explicit

Create a OpenGL texture from image.

This function sends the client-side data (the image) to OpenGL

If img has an invalid size (w or h is 0) or its width or height is bigger than getMaximumSize() then an error is prompted to fg_log and the texture is leaved invalid

imgThe image to send to OpenGL
