Detailed Description

template<class MappedType, class Compare = std::less<MappedType>>
class fg::TextureAtlas< MappedType, Compare >

Class used to manage Texture Atlases

All uploaded textures are associated with a key of type MappedType which with the texture's loaction can be fetced
