template<class T>
enum fm::Transition::Mode

Encode a type of transition.

Enumerator
Linear 

Linear transition { x -> x }.

Smoothstep 

Cubic Hermite interpolation { x -> 3x^2 - 2x^3 }.

Smootherstep 

Smootherstep interpolation { x -> 6x^5 - 15x^4 + 10x^3 }.

Trigonometric 

Trigonometric interpolation { x -> (1 - cos(x*PI))/2 }.
